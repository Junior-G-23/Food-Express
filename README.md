# FoodExpress Professional – 4 Restaurants

## Start
1. ZIP komplett entpacken.
2. Ordner `FoodExpress_Professional_4_Restaurants` öffnen.
3. `index.html` im Browser öffnen.

## Demo Login
Auf `Demo Login` klicken oder selbst registrieren.

## Bilder austauschen
- Hero: `images/hero/hero.jpg`
- Restaurants: `images/restaurants/`
- Gerichte: `images/dishes/`

Beispiel:
Pizza Salami = `images/dishes/salami.jpg`

Wenn du ein Bild ersetzt:
1. Neuer Dateiname muss exakt gleich sein.
2. Browser mit `Strg + F5` neu laden.
